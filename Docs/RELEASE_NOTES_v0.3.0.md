# Swarm-ECS-Sandbox v0.3.0

v0.3.0 completes the static-obstacle safety path and adds a versioned reproduction/diagnostic layer around the deterministic simulation.

## Highlights

### Static obstacle avoidance

- Static OBBs produce stable counter-clockwise obstacle segments.
- Obstacle ORCA half-planes are written before Agent-Agent constraints.
- LP3 receives the actual obstacle-prefix count.
- An immutable BVH prunes both avoidance and swept-collision queries.

### Continuous movement safety

- Fixed-point swept circle vs expanded OBB prevents high-speed tunneling.
- OBB directions quantize to an exact Q16.16 orthonormal basis; outward-rounded AABBs include a proven three-raw-unit truncation bound.
- Circle-corner SAT uses exact raw-coordinate squared distance instead of truncated fixed-point squares.
- Earliest impact uses stable time/obstacle/feature ordering.
- Bounded impact iterations project remaining displacement along contact tangents.
- Circle-vs-OBB/SAT remains a final penetration recovery and diagnostic path.
- Maximum acceleration, turn step and speed limit final velocity changes.

### Replay and diagnostics

- Versioned `.swarmreplay` data uses explicit little-endian encoding, simulation identity/configuration, canonical commands, checkpoints, integrity validation and bounded reads.
- Prevalidated command streams load and play in O(N) total command work; a combined execution budget rejects excessive legal workloads before simulation.
- No-render replay execution supports independent-process comparison.
- Layered authority hashes separate config, world metadata, group state, navigation sequence and Agent component columns.
- First-difference reporting identifies component, entity/group, field and raw expected/actual values.

### Observability

Benchmark and HUD data now distinguish:

- obstacle and Agent ORCA lines;
- avoidance and collision BVH queries/candidates;
- CCD sweep hits and SAT fallback recoveries;
- maximum residual penetration;
- acceleration- and turn-limited Agent counts;
- configuration, full authority and canonical comparison hashes.

## Known limits

- The acceleration/turn limiter runs after holonomic ORCA and may leave the strict ORCA half-plane feasible region. CCD/slide/SAT protect geometry but do not make the result a kinodynamic ORCA optimum.
- Swept collision expands an OBB by Agent radius and uses a slab test. Square expanded corners are conservative and can report contact before a true rounded Minkowski corner.
- Swept broadphase uses the true circular motion AABB and may discard hits that exist only in that square-corner conservative region; real circle/OBB contacts remain covered.
- Static obstacle topology and BVH are immutable within a simulation/rollback epoch.
- Deterministic BVH construction has worst-case `O(N²)` one-time sorting cost; query traversal is worst-case `O(N)` and K-result ordering is `O(K log K)`.
- Headless `Null Device` timing does not measure rendered FPS. Allocation sampling covers the caller thread, not every worker.
- Replay/diagnostics do not constitute a completed runtime/architecture compatibility matrix.
- There is no real UDP/KCP session, authoritative server arbitration, out-of-window snapshot transfer or reconnect protocol in this release.
- Indirect Agent rendering still uploads all instances from CPU and has no per-instance GPU visibility, Hi-Z or HLOD.
- YooAsset/HybridCLR configuration is an integration boundary; production Installer/IL2CPP/bundle/CDN/fallback validation is not included.
